# Personal Portfolio Website

A modern personal portfolio website showcasing my software development projects, technical writing, content writing, and teaching experience.

Designed with a modern **Dark Academia** aesthetic, this website serves as a central hub for my work, allowing visitors to explore my projects, read my articles, and learn more about my skills and experience.

---

## Features

- Responsive modern design
- Dark Academia inspired user interface
- Project showcase (under development)
- Technical writing portfolio (coming soon)
- Content writing portfolio (coming soon)
- Resume download (coming soon)
- Contact information
- GitHub integration
